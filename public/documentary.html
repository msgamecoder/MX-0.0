🚀 How to Use This API

The API returns JSON with keys like content for text or imageUrl for images. It works with fetch, axios, curl, Python requests, bots, and more.

📌 10 Code Examples

1. JS Fetch (Browser)

fetch("https://mxgamecoder-klfx.onrender.com/fun/joke")
  .then(res => res.json())
  .then(data => alert(data.content));

2. Node.js

const https = require("https");
https.get("https://mxgamecoder-klfx.onrender.com/fun/fact", res => {
  let d = "";
  res.on("data", chunk => d += chunk);
  res.on("end", () => console.log(JSON.parse(d).content));
});

3. Curl

curl https://mxgamecoder-klfx.onrender.com/fun/quote

4. Python

import requests
res = requests.get("https://mxgamecoder-klfx.onrender.com/fun/riddle")
print(res.json()["content"])

5. HTML Button

<button onclick="getJoke()">Get Joke</button>

6. React (useEffect)

useEffect(() => {
  fetch("https://mxgamecoder-klfx.onrender.com/fun/quote")
    .then(res => res.json())
    .then(data => console.log(data.content));

7. Discord Bot (Python)

@bot.command()
async def truth(ctx):
  res = requests.get("https://mxgamecoder-klfx.onrender.com/fun/truth").json()
  await ctx.send(res["content"])

8. PHP
